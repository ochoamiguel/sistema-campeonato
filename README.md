# Sistema de Campeonato

**Descrição:**  
Sistema em Python para gerenciar campeonatos de futebol usando SQLite. Permite inserir clubes, consultar estatísticas e calcular automaticamente pontos, saldo de gols e aproveitamento.

---

## Funcionalidades

- Inserir clubes com vitórias, derrotas, empates, gols marcados e sofridos  
- Consulta de clubes por nome, pontos ou aproveitamento  
- Cálculo automático de:
  - Pontos (vitória = 3, empate = 1, derrota = 0)  
  - Saldo de gols  
  - Aproveitamento (%)  
- Menu interativo no console para facilitar o uso

---

## Tecnologias Utilizadas

- Python  
- SQLite (`sqlite3`)
